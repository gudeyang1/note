# git branch
## 1.  创建分支
```
#创建dev分支，但是不切换到新创建的分支
[root@localhost git-test]# git branch
* master
[root@localhost git-test]# git branch dev
[root@localhost git-test]# git branch
  dev
* master
```
## 2. 切换分支
```
#切换到dev分支
gir checkout dev
```
## 3. 合并分支
```
#将dev分支合并到master分支，先切换到master分支
git checkout master
git merge  dev
```
## 4. diff
```
#diff之前需要确保修改的文件已commit
[root@localhost git-test]# git diff master..dev


```
